## Without OpenAI API Key

If you haven't set up the API key, the output looks like this (still useful!):

```
### Tuesday, July 21, 2026

**Commits:** 8 | **Repositories:** 2

Made 8 commits across 2 repositories.
```

The fallback is simple but informative. The AI summaries are nice-to-have but not required!
